从零开始搭建了一个个人站点，用于记录未来所有的成品和项目进展。

- 🪐 星空行星主题
- ⚡ Hugo 静态站点，快速轻量
- 🚀 部署在 GitHub Pages，永久在线
- 📝 Markdown 写作，专注内容

## 技术栈

| 层级 | 选型 |
