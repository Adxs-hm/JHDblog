## 这个项目是什么

**STD Template Mod** 是 RimWorld 1.6 的一个纯 XML 内容模组（零 C# 代码）。作者 **JHD**，packageId `STD.Template.EarlyGame`，中文名"遗落科技"。

核心玩法：殖民者建造一座"STD 分析台"，通过三级递进研究——基础理解 → 进阶解码 → 深度掌握——逐步解锁一系列远超原版的失落科技装备。所有产物仅需两种材料：**钢铁** 和 **重构合金**（50 钢铁在分析台烧制成 1 合金）。

## 设计哲学

这个 Mod 不是堆数值。它有明确的设计原则：

### 1. 功能整合——一件顶多件

原版 RimWorld 中，提升工作速度需要仿生臂、远程攻击需要枪、近战防御需要护盾腰带——三件装备占用三个槽位。STD 的做法是把多种功能**整合到一件装备**中。

例如力场能量枪 MCT-III：远程攻击（44 伤害 × 3 发/40 射程/灼烧效果）**同时**提供 +250% 工作速度加成。一把枪 = 武器 + 仿生臂替代。

护盾腰带 EC-III 更加极端：能量护盾 + 区域照明 + ±200°C 温度调节 + 95% 近战闪避 + 75% 近战减伤 + 工作速度 + 痛阈 + 毒素抵抗 + 免疫速度 + 研究速度。一条腰带整合了原版 6 件装备的功能。

### 2. 理解即力量——研究是瓶颈，不是材料

三个研究等级不单纯提升数值。它们代表殖民者对"模板信号"的理解深度：

- **基础理解**（2 个项目）→ I 级产物——够用，但不出众
- **进阶解码**（3 个项目）→ II 级产物——产出质的飞跃
- **深度掌握**（5 个项目）→ III 级产物——超越游戏平衡，但研究时间极长

总共 10 个研究项目构成完整科技树。瓶颈是**研究时间**而非资源收集——钢铁在 RimWorld 中随手可得。

### 3. 材料极简

不使用任何稀有材料。不要求零部件、高级零部件、超凡材料。只需钢铁 + 重构合金。在 RimWorld 的生态中，这意味着 STD 科技可以由任何发展阶段的殖民地使用——只要你愿意投入研究时间。

## 物品全表

### 重构合金（基础材料）

| 属性 | 数值 |
|------|------|
| 生命值 | ×3（比钢铁耐打） |
| 工作量 | ×0.4（更好加工） |
| 质量 | ×0.6（更轻） |
| 美观度 | ×4（蓝色辉光装饰） |
| 隔热 | ×2 |
| 配方 | 50 钢铁 → 1 合金（在 STD 分析台制作） |

### 力场能量枪 MCT-I/II/III

| 等级 | 伤害 | 弹数 | 射程 | 工作速度加成 | 特色 |
|------|------|------|------|-------------|------|
| I 级 | 22 | ×2 | 28 | +60% | 基础场切割 |
| II 级 | 32 | ×3 | 33 | +120% | 灼烧概率提升 |
| III 级 | 44 | ×3 | 40 | +250% | 重型能量弹 |

三种自定义 DamageDef：`STD_FieldCut`（切割）、`STD_FieldBurn`（灼烧）、`STD_FieldBolt`（穿透能量弹）。

### 护盾腰带 EC-I/II/III

| 等级 | 护盾拦截 | 温控范围 | 近战闪避 | 近战减伤 | 工作速度 |
|------|---------|---------|---------|---------|---------|
| I 级 | ~5 次 | ±50°C | 75% | 40% | +40% |
| II 级 | ~8 次 | ±100°C | 88% | 60% | +80% |
| III 级 | ~12 次 | ±200°C | 95% | 75% | +150% |

附带毒素抵抗、免疫速度、研究速度、痛阈、全局工作速度加成。还有区域照明功能。

### 配给口粮 + 急救包 I/II/III

- 食品营养：2.0 / 2.5 / 3.0
- 心情加成：+18 / +22 / +25
- 医疗效力：2.5 / 4.0 / 5.0
- **永不腐烂**
- 附带代谢优化 Hediff——饥饿速率 -50% / -67% / -80%

急救包和口粮使用同一个配方产出（一份材料同时给两种物品）。

### 永久强化剂 RP-I/II/III

右键使用的消耗品。永久提升全器官能力（意识/移动/操作/泵血/代谢/呼吸/听觉/视觉），III 级超越人类极限。附带**永久伤疤修复**。

### 死亡相关物品（需 Anomaly DLC）

| 物品 | 功能 |
|------|------|
| 模板复生剂 | 对 3 天内遗体使用——复活死者，附带 3 支复苏剂清除副作用 |
| 模板拒死剂 | 对自己使用——死亡时自动复活一次 + 净化副作用 |
| 模板复苏剂 | 清除复活虚弱/精神病/昏迷 |

## 技术实现细节

### XML 结构（9 个 Def 文件）

| 文件 | 内容 |
|------|------|
| `00_Core.xml` | 重构合金 ThingDef + 通用 Abstract 基类 |
| `01_Analyzer.xml` | STD 分析台（Building）+ 配方 |
| `02_Research.xml` | 10 个研究项目，三级递进 |
| `03_MultiCutter.xml` | 力场能量枪三代 + 弹丸 Projectile |
| `05_SupplyPack.xml` | 口粮 + 急救包 + 代谢优化 Hediff |
| `06_EnergyCore.xml` | 护盾腰带三代 + 护盾 Hediff + 全属性加成 |
| `07_WorkGivers.xml` | 双 WorkGiver 绑定（制作 + 烹饪共用一个分析台） |
| `08_RegenPotion.xml` | 永久强化剂三代 + 器官强化 Hediff |
| `09_Resurrection.xml` | 复生剂 + 拒死剂 + 复苏剂（依赖 Anomaly） |

### ⚠️ 已知问题

**P0 缺陷**：About.xml 的 `loadAfter` 只声明了 `Ludeon.RimWorld`，没有声明 Anomaly DLC 依赖。但 XML Def 中实际使用了 Anomaly 专属类（如 `CompUseEffect_Resurrect`）。这会导致没有 Anomaly DLC 的玩家加载 Mod 时崩溃。
