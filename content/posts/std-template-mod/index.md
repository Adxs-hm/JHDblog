## 概述

一个 RimWorld 1.6 的纯 XML 内容 Mod（零 C# 编译依赖）。作者 JHD，packageId `STD.Template.EarlyGame`。核心理念：围绕"STD 分析台"建立三级递进研究体系，解锁一系列超模能量武器、护盾腰带、消耗品和永久强化剂。

## 核心设计哲学

1. **功能整合** — 一件 STD 产物替代 2-4 个原版物品。例如力场能量枪同时提供远程伤害和工作速度加成
2. **理解即力量** — 三个研究等级（基础/进阶/深度）代表殖民者对"模板信号"的理解深度，而非单纯数值提升
3. **材料极简** — 所有物品仅使用钢铁和重构合金（50 钢烧成 1 合金），瓶颈是研究时间而非资源收集

## 物品体系（三级递进 I/II/III）

| 产品线 | 功能 |
|--------|------|
| **重构合金** | 核心材料，钢铁经模板力场重构而成。HPx3，工时x0.4，质量x0.6，美观x4，隔热x2。蓝色辉光 |
| **力场能量枪 MCT** | 能量束远程武器。I 级 22 伤/28 射程；II 级 32 伤/33 射程；III 级 44 伤/40 射程。附带灼烧 + 工作速度加成 |
| **护盾腰带 EC** | 能量护盾 + 区域照明 + 温度调节 + 近战闪避（75-95%）+ 全属性加成 |
| **配给口粮 + 急救包** | 高营养食品（永不腐烂）+ 药品。附带代谢优化（饥饿速率-50%~-80%） |
| **永久强化剂 RP** | 右键使用，永久提升全器官能力，III 级超越人类极限。附带伤疤修复 |
| **模板复生剂/拒死剂/复苏剂** | 死亡相关——复活死者、死亡自动复活、清除复活副作用 |

## 技术特点

- 自定义 3 种能量伤害类型：STD_FieldCut（切割）、STD_FieldBurn（灼烧）、STD_FieldBolt（远程能量弹）
- 10 个研究项目组成三级科技树
- 双 WorkGiver 绑定（制作 + 烹饪共用一个分析台）
- 使用了部分 Anomaly DLC 机制（⚠️ About.xml 尚未声明依赖——已知待修复问题）
- Graphic_Single 渲染模式
