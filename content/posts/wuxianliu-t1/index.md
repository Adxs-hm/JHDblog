## 世界观

### 无限流经典框架

在现实世界濒死或绝望的人会收到一句话：

> *"想要明白生命的意义吗？想要……真正地活着吗？"*
> 【YES】 / 【NO】

选择 YES 的人消失于现实世界，成为"轮回者"；被投入各种任务世界执行高危险任务；成功获得奖励点数和支线剧情；失败则**抹杀**。

### 三层宇宙结构

**第一层：主神广场**
- 无尽白色平台，中央悬浮巨大发光球体——"主神"
- 所有轮回者在此休整、兑换、交流
- 核心功能：兑换系统 / 全身修复 / 造人 / 训练场 / 传送

**第二层：任务世界**
- 150+ 预先设定的世界，覆盖恐怖/科幻/奇幻/历史/武侠/都市
- 分级：E（新人友好）→ D → C → B → A → S（无人生还记录）
- 每个世界有独立评分标准：完成度 / 探索度 / 支线触发 / 隐藏条件

**第三层：团队战役**
- 定期将完整团队投入同一世界相互对抗
- 胜者获得对手的一半资产
- 高风险高回报，鼓励合作但诱发背叛

### 主神广场功能详解

**兑换系统**
- 使用奖励点数 + 支线剧情双货币
- 兑换类别：血统 / 技能 / 功法 / 装备 / 道具 / 特殊
- 高等级兑换需要支线剧情作为资格证明（点数再多也不能跨越等级门槛）

**全身修复**
- 只要没死，主神能修复一切肉体损伤
- 消耗点数，按损伤程度计费
- 无法修复精神/灵魂层面的损伤

**造人**
- 凭空缔造有独立人格的活人
- 可设定年龄/性别/外貌/身体素质/知识储备
- 可设定为绝对忠诚——引发深层伦理问题
- 这是整个无限流设定中最灰暗的功能

**训练场**
- 时间流速可调（1:10 到 1:1000）
- 可用于技能训练、战斗模拟
- 消耗点数，按时间比例计费

## 力量体系

### 六维属性

| 属性 | 作用 | 满值效果（1000点） |
|------|------|-------------------|
| **智力** | 学习速度、魔法威力、精神类技能效果 | 近乎过目不忘，可同时处理多条信息流 |
| **精神力** | 对抗精神攻击、使用精神力技能、造人消耗 | 精神力实质化，可外放形成领域 |
| **细胞活力** | 生命值、自然恢复速度、毒抗 | 断肢再生，百毒不侵 |
| **神经反应速度** | 闪避、速度、远程精度 | 可看清子弹轨迹并躲避 |
| **肌肉组织强度** | 近战伤害、负重、耐久 | 徒手碎钢，单臂举车 |
| **免疫力** | 疾病抵抗、诅咒抗性、异常状态持续 | 几乎免疫一切已知病毒和诅咒 |

**全部六维达到 1,000 点 → 进化为更高维度生命形态（基因锁五阶）。**

### 基因锁系统

| 阶 | 名称 | 效果 |
|----|------|------|
| 一阶 | 战斗直觉 | 无意识中避开致命攻击，肌肉反应速度翻倍 |
| 二阶 | 身体掌控 | 主动调节心率/肾上腺素/痛觉阈值 |
| 三阶 | 大脑解放 | 模拟他人思维模式，短暂预知（3-5 秒） |
| 四阶 | 基因重组 | 主动改造身体形态（如长出翅膀/鳃） |
| 五阶 | 圣人 | 突破人类概念的生物极限，接近规则级 |

### 技能体系（示例）

**东方武学：** 九阳神功（内力恢复×3）、浑天宝鉴（五色真气各有效果）、独孤九剑（破尽天下武学）、降龙十八掌（纯力量输出）

**魔法体系（D&D）：** 从戏法到九环法术，覆盖塑能/咒法/幻术/变化/防护/预言

**型月魔术：** 投影魔术（复制兵器）、强化魔术（属性增幅）、固有结界（展开个人心象世界）

**瞳术：** 写轮眼（动态视力+催眠）、白眼（360°远视）、轮回眼（基础规则级操控引力）

### 双货币经济

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| **奖励点数** | 完成任务、杀死敌人、探索发现、支线事件 | 基础兑换、修复、训练场、日常消耗 |
| **支线剧情** | 任务评价（C→S）、隐藏条件达成、击败 Boss | 解锁高等级兑换的门槛资格 |

这种双货币设计确保了"更多点数 ≠ 能兑换一切"——高等级兑换需要支线剧情作为资格证明，迫使轮回者挑战更高难度任务。

## 特色机制

**任务评分算法**
- 基础完成：D
- 全探索：C
- 触发 2+ 支线：B
- 发现隐藏条件：A
- 达成完美结局：S

**复活机制**
- 队友死亡前获得其身份标记
- 在主神处支付巨额点数（5,000 + B×1 起）
- 复活后附带虚弱期（1-3 个任务世界）
- 必须在任务结束前回收尸体

**团队战规则**
- 两队以上投入同一世界
- 胜者获得败者资产的一半
- 允许投降（投降者失去 30% 资产）
- 禁止在团战中复活队友

### 支线剧情合成

完成隐藏任务或改变剧情可获得支线剧情（D→S 五级）。关键机制：**3 个低一级支线可合成 1 个高一级支线**（3×D = 1×C，以此类推）。高级强化必须消耗对应级别的支线剧情——这是比点数更稀缺的门槛资源。

## 团队系统

### 队长权限

经历足够任务后可挑战"队长试炼"。通过后获得三项特权：(1) 提前获知下一世界名称；(2) 查询隐藏兑换列表；(3) 队长专属团队技能和指挥增益。

### 四种团队定位

| 定位 | 职责 | 典型配置 |
|------|------|---------|
| **近战前锋** | 团队盾牌，正面对抗 | 基因锁使用者、近战血统 |
| **精神控制者** | 扫描/侦查、精神链接、精神遮蔽 | 精神力特长 |
| **火力支援** | 高科技武器/传承火器/远程法术 | 科技或魔法路线 |
| **策略分析师** | 世界谜题破译、战术规划 | 智力特长 |

主神广场中存在**情报交易**市场——老兵可出售已通关世界的完整机制分析，高价值世界情报可换支线剧情。

## 任务世界设计方法论

这是设定包中独有的"元设计"系统——不仅提供了 150+ 个世界设定，更给出了**如何设计新世界**的完整方法论。

### 核心原则

**保留原作锚点，重写一切。** 以《生化危机》为例——病毒、生物武器、保护伞公司是锚点，其余全部重构为原创内容。

### 三明治分层压缩法

| 层级 | 规模 | 内容 |
|------|------|------|
| **骨架层** | ~200 字 | 5 个固定字段：核心悖论、必守规则、危险形态、穿越者身份、隐藏宝藏 |
| **区域层** | ~150 字/区域 | 视觉锚点、活跃威胁、可能性清单 |
| **情报层** | ~100 字/条 | 一句话描述"知道这个能做什么" |

### 四阶段张力引擎

```
锚定期 → 卷入期 → 深化期 → 终局期
```

每个阶段有明确的任务类型、场景限制、压力形式、过渡条件、禁止动作。

## AI 叙事控制原则

- 对 AI 用"禁止项"比"应该项"更有效
- 所有关键信息必须"玩家可感知"
- 阶段转换由 AI 明确宣告
- 每个世界配 5–10 个风味关键词
- 数据卡片按需加载，不一次性全部呈现
