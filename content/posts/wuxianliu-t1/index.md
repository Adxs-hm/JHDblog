## 这是什么

一个经典"主神空间"无限流 **SillyTavern AI 角色扮演设定包**。3 个文件约 499KB，包含约 1,000 条设定（300 血统、376 技能、370 装备、150+ 任务世界、80+ 机制规则）。

这是"酒馆"（SillyTavern）设定包系列中**体量最大**的一个——也是可重玩性最高的一个。

## 世界观

### 经典无限流框架

继承中国网络文学"无限流"流派的经典设定（开创于《无限恐怖》小说）：

- 在现实世界濒死或绝望的人会收到一句话：**"想要明白生命的意义吗？想要……真正地活着吗？"**——【YES】/【NO】
- 选择 YES 的人消失于现实世界，成为"**轮回者**"
- 被投入各种类型的任务世界（恐怖片、科幻、奇幻、历史……），作为小队执行高危险任务
- 成功 → 获得**奖励点数**和**支线剧情**，用于兑换血统/技能/装备
- 失败 → **抹杀**（从存在中彻底消除）
- 死亡 → 真正的死亡（除非有复活道具）

### 三层宇宙结构

| 层级 | 功能 |
|------|------|
| **主神广场** | 无尽白色平台，中央悬浮巨大发光球体——"主神"。所有轮回者在这里休整、兑换、交流 |
| **任务世界** | 150+ 预先设定的世界，从 E 级（新人友好）到 S 级（无人生还记录）。类型覆盖恐怖/科幻/奇幻/历史/武侠/都市 |
| **团队战役** | 定期将完整团队投入同一世界相互对抗。高风险高回报——胜者获得对手的一半资产 |

### 主神广场功能

- **兑换**：点数 + 支线剧情 → 血统/技能/装备/道具
- **全身修复**：只要没死，主神能修复一切肉体损伤（消耗点数）
- **造人**：凭空缔造有独立人格的活人，可设定为绝对忠诚——引发深层伦理问题
- **训练场**：时间流速可调（1:10 到 1:1000），用于训练技能
- **传送**：投入任务世界

## 力量体系

### 六维属性

| 属性 | 作用 |
|------|------|
| **智力** | 学习速度、魔法威力、精神类技能效果 |
| **精神力** | 对抗精神攻击、使用精神力技能、造人消耗 |
| **细胞活力** | 生命值、自然恢复速度、毒抗 |
| **神经反应速度** | 闪避、速度、远程精度 |
| **肌肉组织强度** | 近战伤害、负重、耐久 |
| **免疫力** | 疾病抵抗、诅咒抗性、异常状态持续时间 |

**全部六维达到 1,000 点** → 进化为更高维度生命形态（所谓"解开基因锁第五阶"）。

### 基因锁系统

| 阶 | 效果 |
|----|------|
| 一阶 | 战斗直觉增强，无意识中避开致命攻击 |
| 二阶 | 身体控制力大幅提升，可主动调节心率/肾上腺素 |
| 三阶 | 大脑计算力跃升，可模拟他人思维模式（短暂预知） |
| 四阶 | 基因层面改造自己的身体形态 |
| 五阶 | 圣人——突破人类概念的生物极限 |

### 血统兑换（D → S 级）

- **D 级**：初级狼人、低级吸血鬼（基础属性提升 + 弱点）
- **C 级**：中级血统，有主动技能
- **B 级**：高级血统，显著改变战斗方式（如天使血统的飞行）
- **A 级**：传说级血统（如龙族血统、精灵王血统）
- **S 级**：圣人血统、旧神血统——可改变世界规则

### 技能与功法

- 东方武学：九阳神功、浑天宝鉴、独孤九剑……
- D&D 魔法体系：从戏法到九环法术
- 型月魔术：投影、强化、固有结界
- 瞳术：写轮眼、白眼、轮回眼
- 自定义技能：376 项，涵盖战斗/生存/制作/社交

### 双货币经济

- **奖励点数**：完成任务获得，用于基础兑换
- **支线剧情**：基于任务评价额外奖励，用于解锁高级血统/技能的门槛要求
- 这使得"更多点数"不等于"能兑换一切"——高等级兑换需要支线剧情作为**资格证明**

## 三种叙述者语音

| 模式 | 场景 | 风格 |
|------|------|------|
| 主神广场 | 兑换/修复/等待 | 极简、冰冷、数据化——"兑换：血统-初级狼人。消耗点数：2,000。确认？" |
| 任务世界 | 探索/战斗 | 沉浸式、五感丰富——"你脚下的金属甲板在震动。走廊尽头的红灯以两秒间隔闪烁。空气中有烧焦线路的味道。" |
| NPC 模式 | 对话/交互 | 角色扮演，姓名用「」、动作用 *星号*、内心想法用（括号）区分 |

## 特色机制

- **任务评分系统**：完成度/探索度/支线触发/隐藏条件 → S/A/B/C/D 五级评价
- **复活机制**：死亡后队友可支付巨额点数在主神处复活（必须在任务结束前收回尸体）
- **造人伦理**：被造出的"人"拥有独立人格但被设定为绝对忠诚——这是整个无限流设定中最灰暗的角落
- **团队战**：两队以上被投入同一世界，胜者获得败者资产的一半。鼓励合作但诱发背叛
