## 概述

完全原创的黑暗奇幻穿越世界 **SillyTavern** 角色扮演设定包。872 年前，埃瑟林魔法帝国进行"虚空裂隙计划"实验失败，打开了直通魔界的裂隙。人类被迫用"封魔令"永久封印自身魔力回路以求生存。文明崩塌后仅剩莫瑞干大陆 21 座要塞都市，靠符文枪械、外骨骼装甲和躯体强化改造维生。穿越者主角是 872 年来第一个没有封魔烙印的人——能用古代魔法，且绑定了"命运轮盘"抽卡系统。共 11 个文件约 282KB。

## 世界结构

### 核心设定

| 要素 | 说明 |
|------|------|
| 莫瑞干大陆 | 仅存的大陆，21 座要塞都市 + 数百乡镇 + 无数小型定居点 |
| 封魔令 | 永久封印人类魔力回路的烙印术式——魔力会吸引魔物 |
| 魔界 | 裂隙直通的异世界，魔物从低阶到"万王之王" |
| 魔物十二阶 | 觉醒位→淬体位→……→魔王（十一二阶）→万王之王 |
| 猎魔军 | 按贡献值而非个人实力评定的军事体系 |

### 核心机制

- **命运轮盘** — 四池抽卡系统（武器/技能/知识/特殊），使用击杀魔物获得的命运点数
- **三大配置方案** — 仅世界书 / 纯角色卡 / 混合方案（推荐）
