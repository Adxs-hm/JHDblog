## 世界观：封魔令的悲剧

### 埃瑟林魔法帝国的覆灭

872 年前，埃瑟林魔法帝国的十二高塔议会实施了"虚空裂隙计划"——撕裂空间壁垒，从虚空中提取无限能量。

**实验失败了。** 裂隙直接连通了**魔界**。

十二位至高大法师中，十一人在强行关闭裂隙时身亡。仅剩的**泰瑞斯·封魔者**——在目睹十一人牺牲后——做出了一个改变文明走向的决断：颁布"封魔令"。

### 封魔令悖论

封魔令是一种永久封印人类魔力回路的烙印术式。不是暂时压制——是**永久性切断**。因为魔力本身就是吸引魔物的信标。

**代价**：一个以魔法称霸的文明，被迫抛弃了自己赖以称霸的一切。

**结果**：帝国覆灭。文明崩塌。八百七十二年来，人类靠着"战争科技"——基于魔晶的符文枪械、外骨骼装甲和躯体强化改造——勉强维生。

这是封魔令悖论的核心：**封印力量以求生存，但封印了力量之后，还能称为"生存"吗？**

### 穿越者

穿越者主角是 872 年来**第一个没有封魔烙印的人类**。魔力回路完整。能使用古代魔法。

同时还绑定了一个"命运轮盘"——四池抽卡系统。使得穿越者在物理上变成了人类对抗魔界的终极武器。

但问题来了：**如果摆脱封魔令，魔法就会重新吸引魔物——魔界的威胁量级会重新升级。人类在 872 年里建立的"战争科技"体系，在魔法级别的对抗中不堪一击。**

穿越者是救星，还是灾厄？

## 莫瑞干大陆

## 主题

- **代价**：封魔令是人类为生存支付的终极代价——自断双臂，只求不被发现
- **选择的重量**：穿越者解除封印 vs 让人类继续依赖战争科技——两种未来都有人会死
- **在废墟中重建的意义**：872 年来人类一直在"活下去"和"活得像个人"之间挣扎

## 战争科技体系

封魔令切断魔力后，人类用 872 年发展出五类替代技术。古代魔法是个人/内部/不可复制的——战争科技是**工业/外部/可量产的**。

| 技术 | 核心 | 指标 |
|------|------|------|
| **躯体强化公式** | 金属骨架 + 外骨骼，魔晶供能 | 高级版反应速度肉眼不可见，但肉体损伤极大 |
| **魔焰枪** | 嵌入符文阵列 | 射程 150m，熔化钢材 |
| **符文弹机枪** | 微型攻击符文印在弹头上 | 穿甲/爆炸/冰冻/腐蚀可选 |
| **符文装甲** | 多层防护符文叠加 | 可承受六阶魔物直击 |
| **魔械结合** | 蒸汽机改风符文，攻城器减重力法术 | 人力极限的百倍 |

## 魔晶系统

魔物死亡后体内能量失去宿主约束，在数秒至数分钟内结晶化——这便是魔晶的由来。

### 三大学说

| 学说 | 观点 | 伦理含义 |
|------|------|---------|
| 能量晶体论（主流） | 安全储能，无意识残留 | 用完即弃 |
| 灵魂囚笼论（少数派） | 晶体含魔物意识，使用即消耗灵魂 | 有伦理负担 |
| 世界碎片论（极端少数） | 每颗晶体是魔界本身的一块碎片 | 本质是蚕食两个世界 |

### 七级晶体

| 等级 | 来源 | 用途 |
|------|------|------|
| 灰（1–2阶） | 民间 | 日常供能 |
| 白（3–4阶） | 制式装备 | 符文装甲 1 小时 |
| 蓝（5–6阶） | 高级装备 | 主力军用 |
| 紫（7–8阶） | 城防级 | 要塞护盾 1 日 |
| 金（9–10阶） | 唯一属性 | 火焰/冰霜/雷电特化 |
| 黑（王级） | 仅 3 颗记录 | 可锻造"弑王"兵器 |
| 黑王 | 未见 | 可能不存在 |

未净化晶体的危险：短期——头痛、幻觉、情绪不稳；长期——躯体变异、鳞角增生、丧失人性；直接吸收——变为"魔化人"（保留智力 + 魔物杀戮本能）。净化需专用熔炉，蓝晶 ~3 天，金晶 ~3 个月，能量损失 15–30%。

## 魔界生态

魔界是与人类世界并行的行星——魔法浓度 3,000 倍，"黑日"只散发暗紫色辐射，植物从土壤中吸收魔法（紫蓝红色），重力 1.2×，氧气更低。

### 四大智能层级

```
野兽魔物 → 智慧魔物 → 魔王 → 远古魔物（仅万王之王确认）
```

六大形态：兽型 / 人形 / 复合型 / 无形 / 寄生型 / 拟态型。

### 吞噬进化

四道关卡：(1) 觉醒（野兽→智慧，90% 无法跨越）；(2) 领域（7阶+，展开改变局部法则的魔力场）；(3) 王化（10阶+，免疫衰老）；(4) 降临（王→世界级——需吞噬 3 个同级王……或一整个世界碎片）。

## 传送门真相

传送门是**双向**的——人类也能穿越到魔界。灰塔已完成 3 次侦察任务。虚空石（传送门锚点）只能在魔界制造——人类无法"封印"传送门，只能摧毁它。

**世纪潮现象：** 自第 756 年起，传送门密度增加了 7 倍，并持续上升。三大假说：万王之王苏醒、两界正在融合、人类魔法复燃的反馈。
