STD Template Mod（遗落科技）

RimWorld 1.6 纯 XML 失落科技模组——三级递进研究、功能整合设计哲学、零 C# 编译依赖
June 30, 2026 · JHD

这个项目是什么#

STD Template Mod 是 RimWorld 1.6 的一个纯 XML 内容模组（零 C# 代码）。作者 JHD，packageId STD.Template.EarlyGame，中文名"遗落科技"。

核心玩法：殖民者建造一座"STD 分析台"，通过三级递进研究——基础理解 → 进阶解码 → 深度掌握——逐步解锁一系列远超原版的失落科技装备。所有产物仅需两种材料：钢铁 和 重构合金（50 钢铁在分析台烧制成 1 合金）。

“STD"命名源于设定中"Standard Template Data”（标准模板数据）的概念——一种失落文明遗留下来的技术编码格式。分析台就是解码这些模板的设备。

游戏体验流程#

以一个典型的新殖民地为例，STD 科技的体验路线：

阶段 1：发现与初探（研究等级 1-2）

  • 殖民者首先研究"模板信号解析"，解锁 STD 分析台的建造配方
  • 建造分析台后，可以将 50 钢铁转化为 1 重构合金——合金带有蓝色辉光，美观度 ×4，本身就是很好的建材和装饰品
  • 继续研究"基础力场应用"，解锁 MCT-I 力场能量枪和 EC-I 护盾腰带
  • 此时殖民者获得一把比原版突击步枪更强（28 射程 vs 31，但 22×2 伤害且带灼烧）的武器，外加一条 +60% 工作速度的腰带

阶段 2：进阶应用（研究等级 3-5）

  • 解锁 II 级装备：MCT-II（32×3 发/33 射程/灼烧/120% 工速）、EC-II（8 次护盾拦截/±100°C 温控/88% 近战闪避/80% 工速）
  • 解锁配给口粮和急救包的生产线——一个配方同时产出食物和药物，口粮永不腐烂
  • 此时一个装备 II 级护盾腰带的殖民者在极端气候下几乎不需要额外的温度管理

阶段 3：深度掌握（研究等级 6-10）

  • III 级装备解锁，各项数值远超原版任何装备
  • 永久强化剂 RP-III 可将殖民者的全器官能力提升至超越人类的水平
  • 模板复生剂和拒死剂提供原版最稀缺的资源——“第二次机会”
  • 此时 STD 科技殖民地的战斗力约等于原版的 3-4 倍

设计哲学#

1. 功能整合——一件顶多件#

原版 RimWorld 中，提升工作速度需要仿生臂、远程攻击需要枪、近战防御需要护盾腰带——三件装备占用三个槽位。STD 的做�是把多种功能整合到一件装备中。

护盾腰带 EC-III 功能拆解：

功能原版等效装备/方式原版槽位
能量护盾（~12 次拦截）护盾腰带腰带
区域照明落地灯 / 黑暗环境不便建筑
±200°C 温度调节防寒/防热衣物各一套外层+中层
95% 近战闪避高品质护盾腰带 + 敏捷特性-
75% 近战减伤海军装甲 + 高品质外层+中层
+150% 工作速度仿生臂 ×2 + 工作勤奋特性-
痛阈/毒素/免疫/研究各类仿生器官 + 药物多个槽位

一条 EC-III 腰带 ≈ 原版 6-8 件装备的效果总和，且只占用一个腰带槽位。

2. 理解即力量——研究是瓶颈#

研究等级项目数典型研究耗时解锁
基础理解2~3,000 点I 级装备，合金配方
进阶解码3~8,000 点II 级装备，口粮/急救包
深度掌握5~25,000 点III 级装备，永久强化剂，复活系列

10 个研究项目总计约 36,000 研究点——相当于原版从石器时代到太空时代的全部研究点总和。瓶颈是研究时间而非资源收集。

3. 材料极简#

不使用任何稀有材料。不要求零部件、高级零部件、超凡材料。任何发展阶段的殖民地都能使用 STD 科技——只要你愿意投入研究时间。这使得 STD Mod 在"低科技开局"的玩法中特别有价值。

物品详细规格#

重构合金（基础材料）#

合金是在分析台中通过"力场重构"工艺制造的——模板信号改变钢铁的晶体结构，产物带有蓝色辉光。

  • STD_FieldBolt（场能弹）——能量弹丸，穿透力强，对装甲有加成

代谢优化 Hediff（STD_MetabolicOptimization）通过降低饥饿速率来延长食物供给的有效时间——III 级可将单人口粮维持时间从 1.5 天延长至 7.5 天。

永久强化剂 RP-I/II/III#

右键使用的消耗品，永久提升全器官能力。强化的器官包括：大脑（意识）、眼（视觉）、耳（听觉）、心（泵血）、肝（代谢）、肾（过滤）、肺（呼吸）、胃（消化）、臂（操作）、腿（移动）。

等级单项加成伤疤修复
RP-I+15%无
RP-II+30%新伤疤自动修复

III 级强化剂的最终效果：意识 150%、移动 150%、操作 150%、泵血 150%、代谢 150%、呼吸 150%。这个数值在游戏机制中已经超越了"人类"的生物学极限。

死亡相关物品#

物品机制使用条件

复活机制完全依赖 Anomaly DLC 的 CompUseEffect_Resurrect 类和 DeathRefusal 机制。这是 About.xml 中未声明 Anomaly 依赖的 P0 缺陷的原因。

技术实现#

XML 架构模式#

Abstract 基类继承：

text
ThingDef_Base (Abstract)
  ├── STD_Alloy (继承 Base)
  │     └── stats: HP×3, Work×0.4, Mass×0.6, Beauty×4, Insulation×2
  ├── STD_Weapon_Base (Abstract, 继承 Base)
  │     ├── MCT-I (具体武器参数)
  │     ├── MCT-II
  │     └── MCT-III
  ├── STD_Apparel_Base (Abstract, 继承 Base)
  │     ├── EC-I (具体腰带参数)
  │     ├── EC-II
  │     └── EC-III
  └── STD_Ingestible_Base (Abstract, 继承 Base)
        ├── Supply-I
        ├── Supply-II
        └── Supply-III

这种继承结构将公共属性（如所有 III 级物品所需的研究前提、所有能量武器的伤害类型）抽取到基类中，减少重复代码约 40%。

贴图管线#

  1. Blender 建模：在 Blender 中建立精确的 3D 模型（分析台、武器、护盾腰带），使用 Cycles 渲染为高分辨率 PNG（1024×1024）
  2. AI 风格转换：使用 AI 工具将渲染图统一转换为 RimWorld 原版的水彩/手绘风格
  3. Python 批处理：PIL 脚本裁剪为 256×256、转换为 PNG、应用到正确的命名规范（STD_FieldGun.png、STD_ShieldBelt.png 等）
  4. 贴图验证：在游戏内检查 UI 图标大小（256×256 最佳）和世界渲染效果

与终局世界观的关联#

“STD”（Standard Template Data）概念与「终局时代」设定中的"模板"（Template）同源——都是对本体信息的标准化编码。Mod 中的"模板信号解码"研究，在设定层面对应终局时代中人类重新发现"元模板"（Meta-Templates）的过程。

这不是一个单纯的游戏 Mod——它是「终局时代」世界观在 RimWorld 机制层面的一个具象化实验。
