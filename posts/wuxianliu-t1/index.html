世界观#

无限流经典框架#

在现实世界濒死或绝望的人会收到一句话：

“想要明白生命的意义吗？想要……真正地活着吗？” 【YES】 / 【NO】

选择 YES 的人消失于现实世界，成为"轮回者"；被投入各种任务世界执行高危险任务；成功获得奖励点数和支线剧情；失败则抹杀。

三层宇宙结构#

第一层：主神广场

  • 无尽白色平台，中央悬浮巨大发光球体——“主神”
  • 所有轮回者在此休整、兑换、交流
  • 核心功能：兑换系统 / 全身修复 / 造人 / 训练场 / 传送

第二层：任务世界

  • 150+ 预先设定的世界，覆盖恐怖/科幻/奇幻/历史/武侠/都市
  • 分级：E（新人友好）→ D → C → B → A → S（无人生还记录）
  • 每个世界有独立评分标准：完成度 / 探索度 / 支线触发 / 隐藏条件

第三层：团队战役

  • 定期将完整团队投入同一世界相互对抗
  • 胜者获得对手的一半资产
  • 高风险高回报，鼓励合作但诱发背叛

主神广场功能详解#

兑换系统

  • 使用奖励点数 + 支线剧情双货币
  • 兑换类别：血统 / 技能 / 功法 / 装备 / 道具 / 特殊
  • 高等级兑换需要支线剧情作为资格证明（点数再多也不能跨越等级门槛）

全身修复

  • 只要没死，主神能修复一切肉体损伤
  • 消耗点数，按损伤程度计费
  • 无法修复精神/灵魂层面的损伤

造人

  • 凭空缔造有独立人格的活人
  • 可设定年龄/性别/外貌/身体素质/知识储备
  • 可设定为绝对忠诚——引发深层伦理问题
  • 这是整个无限流设定中最灰暗的功能

训练场

  • 时间流速可调（1:10 到 1:1000）
  • 可用于技能训练、战斗模拟
  • 消耗点数，按时间比例计费

双货币经济#

货币获取方式用途
奖励点数完成任务、杀死敌人、探索发现、支线事件基础兑换、修复、训练场、日常消耗
支线剧情任务评价（C→S）、隐藏条件达成、击败 Boss解锁高等级兑换的门槛资格

这种双货币设计确保了"更多点数 ≠ 能兑换一切"——高等级兑换需要支线剧情作为资格证明，迫使轮回者挑战更高难度任务。

三种叙述者语音#

模式场景风格示例
主神广场兑换/修复/等待“兑换：血统-初级狼人。消耗点数：2,000。支线剧情：D×1。确认？”
任务世界探索/战斗“你脚下的金属甲板在震动。走廊尽头的红灯以两秒间隔闪烁。空气中弥漫着烧焦线路的味道。”
NPC 模式对话/交互*「你也是这次的新人？」教官打量着你一眼，手指敲了敲桌面。他显然不打算给你太多思考的时间。

特色机制#

任务评分算法

  • 基础完成：D
  • 全探索：C
  • 触发 2+ 支线：B
  • 发现隐藏条件：A
  • 达成完美结局：S

复活机制

  • 队友死亡前获得其身份标记
  • 在主神处支付巨额点数（5,000 + B×1 起）
  • 复活后附带虚弱期（1-3 个任务世界）
  • 必须在任务结束前回收尸体

团队战规则

  • 两队以上投入同一世界
  • 胜者获得败者资产的一半
  • 允许投降（投降者失去 30% 资产）
  • 禁止在团战中复活队友

支线剧情合成#

完成隐藏任务或改变剧情可获得支线剧情（D→S 五级）。关键机制：3 个低一级支线可合成 1 个高一级支线（3×D = 1×C，以此类推）。高级强化必须消耗对应级别的支线剧情——这是比点数更稀缺的门槛资源。

队长权限#

经历足够任务后可挑战"队长试炼"。通过后获得三项特权：(1) 提前获知下一世界名称；(2) 查询隐藏兑换列表；(3) 队长专属团队技能和指挥增益。

主神广场中存在情报交易市场——老兵可出售已通关世界的完整机制分析，高价值世界情报可换支线剧情。

任务世界设计方法论#

这是设定包中独有的"元设计"系统——不仅提供了 150+ 个世界设定，更给出了如何设计新世界的完整方法论。

核心原则#

保留原作锚点，重写一切。 以《生化危机》为例——病毒、生物武器、保护伞公司是锚点，其余全部重构为原创内容。

三明治分层压缩法#

层级规模内容
骨架层~200 字5 个固定字段：核心悖论、必守规则、危险形态、穿越者身份、隐藏宝藏
区域层~150 字/区域视觉锚点、活跃威胁、可能性清单
情报层~100 字/条一句话描述"知道这个能做什么"

AI 叙事控制原则#

  • 对 AI 用"禁止项"比"应该项"更有效
  • 所有关键信息必须"玩家可感知"
  • 阶段转换由 AI 明确宣告
  • 每个世界配 5–10 个风味关键词
  • 数据卡片按需加载，不一次性全部呈现
